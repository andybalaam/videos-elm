type alias Model = { email1 : String , email2 : String } model = { email1 = "" , email2 = "" }

Clean

view address model = ... div [] [ button [ submitDisabled model ] [ text "Submit" ] ] ...

Clean

submitDisabled model = disabled ( validEmail model.email1 && model.email1 == model.email2 ) validEmail string = contains "@" string

Clean

type Action = Email1Changed String | Email2Changed String update action model = case action of Email1Changed s -> { model | email1 = s } Email2Changed s -> { model | email2 = s }

Clean

  • Everything you care about is in the Model
  • Everything you see is in view
  • Everything that can happen is in update
  • No more guesswork
